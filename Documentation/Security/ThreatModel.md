## Login

- **Ameaças:**
- Ataque de força bruta
- injeção de SQL
- Vazamento de credenciais
- **Mitigações:**
- Bloqueio temporário após múltiplas tentativas falhas
- Validação e sanitização de inputs
- Hash de senhas e armazenamento seguro

## Agendamento

- **Ameaças:**
- Alteração não autorizada de compromissos
- Acesso a dados de outros usuários
- **Mitigações:**
- Controle de acesso baseado em roles (RBAC)
- Logs de auditoria de alterações 
- Validação de propriedade dos dados antes de alterações
